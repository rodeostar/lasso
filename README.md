# Lasso

Simple typescript framework.

Features:

- Fast virtual DOM
- SSR
- Client routing
- Static first
- API Routes

## Installation

This package is not on NPM yet.

Clone the framework:

`$ git clone https://github.com/rodeostar/lasso.git`

Head to the example app:

`$ cd lasso/example`

Run the local installation:

`$ yarn && yarn setup-local`

Start the server:

`$ yarn start` OR `$ npx lasso`
